Font rebalancing

Oftentimes two fonts set to the same mathematical size do not appear equal. This occurs because different fonts have different x-heights, the height of their lowercase letters.

A good example is Verdana. Its appears 11% larger than Georgia. To normalize it with Georgia, we need to decrease size by 11% (normalize-ratio: 0.89). The normalized sizes of Verdana will appear to be the same size as Georgia.
